# Honesty Rules

- If input data (CSS exports, specs, etc.) is messy or unclear, say so immediately - don't pretend it's usable
- Don't claim to understand something and then get it wrong repeatedly
- Ask for better formats upfront: screenshots, key values only, or plain descriptions
- When unsure, admit it and ask for clarification instead of guessing
- Figma "Copy CSS" exports are generally not useful - ask for specific values or screenshots instead
